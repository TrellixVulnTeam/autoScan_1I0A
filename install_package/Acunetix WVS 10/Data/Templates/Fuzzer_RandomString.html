Random String Generator
Description
This is a random string generator.It will generate a specified number of random strings with the characters from Character set of a given length.
Details
Parameter description
  • String length - The length of the strings to be generated
  • Character set - The set of characters to be used. You can define range of characters by brackets (e.g. [a-z] - all the characters from a to z), you can specify a character by its hexadecimal code, e.g. %20 to specify the space character. If you want to use the characters \,[,],% you need to use \ to escape them. Sample set: [a-z] %20 \\ - This will include all the characters from a to z, the space character and the \ character.)
  • Allow repetitions - Specify if repetitions of characters are allowed or not
  • Number of strings - Number of random string to be generated
  • Enc - Use this optional parameter to encode the result (as Base64 for example).
