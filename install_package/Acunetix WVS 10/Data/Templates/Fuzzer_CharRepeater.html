Character Repeater
Description
This is a character repeater generator. It will repeat a specified character/string for a given number of times. This is commonly used for buffer overflow testing.
Details
Parameter description
  • Character/String - Which character or string to be repeated
  • Initial count , Final count - Initial and final repetition count
  • Increment - The increment to be used (default is 1)
  • Enc - Use this optional parameter to encode the result (as Base64 for example).
